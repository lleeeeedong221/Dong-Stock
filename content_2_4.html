기관,외인들은 어떤 놈들이누?

수요와 공급을 알아야 호갱을 안당할 수 있다. ㅎㅎ 그러면 어떤 놈들이 수요와 공급을 제공하는지 알아보까?

크게 4놈들이 있다. 외국인,기관,개인,프로그램.

1.외국인 : 그냥 외국인 증권사임. 외놈들

2.개인 : 너

3.기관 : 기관을 좀 자세히 알아보자. 기관에는 여러 기관들이 있는데
A.투신 : 투자신탁의 약자. 즉, FUND라고 생각하면 된다.
B.금융투자 : 우리가 아는 증권회사들
C.보험 : 걍 보험
D.은행 : 걍 은행
E.기타금융 : 새마을금고연합회 등 기타 금융기관
F.연기금 : 연금기금, 국가투자채에서 운용하는 자금을 말한다. 이놈은 600조라는 막대한 돈을 굴리는 놈. 예의주시하자.
G.사모펀드 : 몰라두댕 ㅎ ( 사적으로 모인 투자 기금 )
H.기타법인 : 뭐꼬
I.기타외국인 : 외노자.

4.프로그램 : 선물에 따른 차익매매 ( 선물가격과 현물가격의 차이를 이용해서 매매 ), 비차익매매 ( 여러종목을 뭉탱이로 산다. )로 나뉜다.
